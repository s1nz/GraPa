# GraPa
### Subtle CSS Gradient Patterns

## [See Demo](http://simurai.github.com/GraPa/)

#### How to use

```css
@import "grapa.less";
.css-class { 
    .grapa-[nr]( color, size ); 
}
```

For example:

```css
@import "grapa.less";
.css-class { 
    .grapa-5( hsl(0,0%,95%), 30px ); 
}
```
